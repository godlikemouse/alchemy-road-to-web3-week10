# Alchemy's Road to Web3 Week 10

This repository covers the modified code following Alchemy's Road to Web3 Week 10.

## Development Setup

Change directory to `./road-to-lens` and execute the following:

    npm i

## Running

Change directory to `./road-to-lens` and execute the following:

    npm run dev
